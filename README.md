# BlastOff Preact

Reusable BlastOff components for Preact

```sh
$ yarn add @stomprocket/blastoff-preact
```
```js
import { Navbar } from '@stomprocket/blastoff-preact'
let links = [
    {
        href: "/foo",
        body: "Foo"
    }
]
// ...
    <Navbar title="Nav Title" links={links} />
// ...
```
